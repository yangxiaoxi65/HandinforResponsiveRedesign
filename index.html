Responsive Redesign Project: Ceremony Providence

Project Overview

This project focused on redesigning the Ceremony Providence location page to ensure it is fully responsive and accessible, providing a seamless user experience across various devices.

Context & Purpose

The Ceremony Providence's location page, https://www.ceremonypvd.com/location, was chosen for redesign because it presented several opportunities for improving efficiency, usability, and accessibility. Despite offering essential details such as the cafe's address, business hours, and a brief introduction, the page lacked an interactive map, clear navigation cues, and responsive design, making it less engaging and accessible, especially on smaller screens.

Original Website Screenshot

Reasons for Choosing This Website

  • Insufficient responsive design leading to a confusing navigation experience on smaller screens.
  • Lack of navigational efficiency and clarity, with the navigation bar not clearly indicating the current page.
  • Inadequate accessibility features, with images, buttons, and links missing crucial alt and aria-label tags.
  • Essential information like the integrated map and contact details not prominently displayed, hindering user interaction.
